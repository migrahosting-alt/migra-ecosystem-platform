## MigraHosting Social Pack

Editable SVG social graphics for MigraHosting in the premium neon hosting style shown in the reference examples.

### Included formats

- `landscape/mh-website-launch-landscape-v1.svg` - 1200x628
- `instagram/mh-managed-cloud-instagram-v1.svg` - 1080x1350
- `portrait/mh-30-day-guarantee-portrait-v1.svg` - 1080x1350
- `story/mh-wordpress-story-v1.svg` - 1080x1920

### Notes

- All files are standalone SVGs with no external image dependencies.
- Text is editable directly in any SVG-friendly design tool or code editor.
- Fonts referenced: `Sora`, `Space Grotesk`, with `Arial, sans-serif` fallback.
- Palette direction: deep navy, violet, magenta, electric cyan, warm gold accents.

### Suggested usage

- `landscape`: X, LinkedIn, ads, blog headers
- `instagram`: Instagram feed portrait, Facebook feed portrait
- `portrait`: promotional feed posts
- `story`: Instagram Stories, Facebook Stories, vertical ads
