# Task: [Feature Name]

## Objective
Build a complete, visible, usable feature.

## Requirements

### 1. UI
- Create page at: [route]
- Add navigation entry
- Display real or empty state

### 2. Functionality
- Implement main user action: [describe]

### 3. Data
- Connect to API
- Persist changes
- Reflect updates in UI

### 4. UX
- Add loading state
- Add empty state
- Add error state

## Visibility Requirement
After implementation, an `OWNER` must be able to:

- see the feature
- access it via navigation
- perform the main action
- see the result immediately

## Forbidden
- backend-only work
- hidden routes
- placeholder UI
- "later" implementations

## Definition of Done
- page exists
- navigation exists
- feature is usable
- data is visible
- interaction works end-to-end

## Output Required
1. what page was built
2. what user can do now
3. what is visible in UI
