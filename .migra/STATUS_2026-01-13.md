### Features

- ✅ **Fallback**: Uses local cache if vault unreachable
- ✅ **Hot-reload**: Services reload config without restart
- ✅ **Encrypted**: AES-256-GCM encryption at rest
- ✅ **Fault-tolerant**: Survives vault outages using TTL-based cache
- ✅ **Audit trail**: All secret access logged

### Usage in mPanel
